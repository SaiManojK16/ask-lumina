# Lumina Screens Assistant

## Overview

A smart assistant application powered by OpenAI to assist customers with home theater setup recommendations, FAQs, and product details from Lumina Screens.

## Technologies
- **Frontend**: Next.js, Tailwind CSS
- **Backend**: OpenAI API
- **AI**: GPT-4 for AI-based responses

## Features
- Real-time conversational AI
- Home theater setup guidance
- Product recommendations
- FAQ answering for Lumina Screens

## Installation
1. Clone the repository
2. Run `npm install`
3. Set environment variables in `.env.local`
4. Run `npm run dev` to start the app
